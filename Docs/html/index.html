Hopman
A 2D platformer by Eric VandeGriek
Hopman Documentation Main Page

A 2D platformer using SDL2

Features:

  • Parallax scrolling background
  • Multiple enemy types
  • Animated sprites
  • A level editor
  • A menu system

Controls:

  • Use the arrow keys or A and D to move left and right
  • Press Space to jump
  • Press Q to quit
  • Esc opens pause menu
  • The F key toggles an FPS display

The main game class is Hopman
